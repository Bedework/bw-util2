Introduction

This project provides a number of network (http, DAV,servlet etc) related classes and methods for Bedework.

1. Util-dav

Provides a utility class to interact with DAV servers.

1.1. Using this project

Add the artifact(s) as a dependency to your project:

      <dependency>
        <groupId>org.bedework</groupId>
        <artifactId>bw-util-dav</artifactId>
      </dependency>

2. Util-http

Provides utility classes to handle http interactions.

2.1. Using this project

Add the artifact(s) as a dependency to your project:

      <dependency>
        <groupId>org.bedework</groupId>
        <artifactId>bw-util-http</artifactId>
      </dependency>

3. Util-servlet

Provides classes and interfaces to help build servlets.

3.1. Using this project

Add the artifact(s) as a dependency to your project:

      <dependency>
        <groupId>org.bedework</groupId>
        <artifactId>bw-util-servlet</artifactId>
      </dependency>

4. Util-servlet-filters

Provides base classes and abstract implementations for servlet filters.

4.1. Using this project

Add the artifact(s) as a dependency to your project:

      <dependency>
        <groupId>org.bedework</groupId>
        <artifactId>bw-util-servlet-filters</artifactId>
      </dependency>
